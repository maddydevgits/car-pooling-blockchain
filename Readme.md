# About Project

Peer to Peer Car Pooling is a project on Blockchain to introduce the concepts of car pooling using smart contracts and the context of blockchain.

# Development Process
1. Initialise Truffle  :heavy_check_mark:
2. Create Register and Login for the project :heavy_check_mark:
3. Create a dashboard page :heavy_check_mark:
4. Create a ride :heavy_check_mark:
5. Bid the ride :heavy_check_mark:
6. Accept the high costed ride :heavy_check_mark:
7. Create Smart Contracts for Car Pooling :heavy_check_mark:

# Softwares
1. Truffle
2. NodeJs
3. Ganache
4. Python3

# Packages
1. flask
2. web3

# Execution Process
1. Clone the repo
2. Launch the Ganache
3. Compile the Smart Contract (truffle compile)
4. Migrate the Smart Contract (truffle migrate)
5. Copy the Smart Contract Address into ca.py
6. Run the server (python app.py)
